# CaneNext : Online Signature System

เวอร์ชันนี้เพิ่มการกรองข้อมูล 2 ระดับ:

- เขตหลัก: อ้างอิงจากตัวเลข 2 ตัวแรกของชื่อไฟล์ เช่น `01_1234_เอกสาร.pdf` → เขต `01`
- เขตย่อย: อ้างอิงจากตัวอักษรลำดับที่ 4 ถึง 7 ของชื่อไฟล์ เช่น `01_1234_เอกสาร.pdf` → เขตย่อย `1234`

## สิ่งที่ปรับเพิ่ม

- เพิ่มตัวกรอง `เขตย่อย`
- รายการเขตย่อยจะสร้างจากไฟล์ PDF ที่มีอยู่จริงใน Google Drive
- เมื่อเลือกเขตหลัก รายการเขตย่อยจะปรับตามเขตที่เลือก
- สามารถกรองร่วมกันได้: สถานะเอกสาร + เขตหลัก + เขตย่อย + ค้นหาชื่อไฟล์
- จำนวนในแท็บ ทั้งหมด / เซ็นแล้ว / ยังไม่เซ็น จะเปลี่ยนตามเงื่อนไขที่กรอง

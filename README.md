# CaneNext : Online Signature System

เวอร์ชันนี้ปรับหน้าสรุปสถานะเอกสารให้แสดงเป็นตารางตามเงื่อนไขที่กำหนด:

- หากยังไม่เลือกเขต: แสดงตารางสรุปรายเขตเรียงจาก 01 ถึง 12
- หากเลือกเขต: แสดงตารางสรุปรายเขตย่อยของเขตนั้น
- เพิ่มตัวกรองเขตและเขตย่อยในหน้าสรุปสถานะเอกสาร
- เขตหลักอ้างอิงจากตัวเลข 2 ตัวแรกของชื่อไฟล์
- เขตย่อยอ้างอิงจากตัวอักษร/ตัวเลขลำดับที่ 4 ถึง 7 ของชื่อไฟล์

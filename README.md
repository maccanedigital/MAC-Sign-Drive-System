# CaneNext : Online Signature System

อัปเดตครั้งนี้เพิ่มตัวกรองเขตย่อยและปรับตารางสรุปสถานะเอกสาร:

- เขตหลักอ้างอิงจากตัวเลข 2 ตัวแรกของชื่อไฟล์ เช่น `01_1234_เอกสาร.pdf` = เขต `01`
- เขตย่อยอ้างอิงจากตัวอักษรลำดับที่ 4 ถึง 7 ของชื่อไฟล์ เช่น `01_1234_เอกสาร.pdf` = เขตย่อย `1234`
- หากยังไม่เลือกเขต จะแสดงตารางสรุปรายเขต `01–12`
- หากเลือกเขต จะแสดงตารางสรุปรายเขตย่อยของเขตนั้น
- รายการเอกสาร PDF สามารถกรองร่วมกันได้ทั้ง เขตหลัก / เขตย่อย / สถานะ / คำค้นหา
